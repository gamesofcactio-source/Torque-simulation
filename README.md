# Torque Simulation
Simulating torque in 2D space using SDL library in C++

Feel free to clone the repository on you system 
```
https://github.com/gamesofcactio-source/Torque-simulation.git
```

Generate and build
```
cd Torque-simulation
mkdir build
cmake ..
cmake --build .
```

Run the game
```
./torque-simulation
```
